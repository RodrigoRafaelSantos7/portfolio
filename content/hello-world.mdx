# Aula 3

## Formular um problema de procura

#### Porque procurar?

Temos um problema difícil. Dada uma descrição que nos ajuda a reconhecer uma solução, sem nos ser dado um algoritmo que resolve o problema, temos que procurar por uma solução.

#### Definição de um problema de pesquisa

Um problema de pesquisa é definido por:

- Um conjunto de estados
- Um estado inicial
- Estados objetivo ou um teste objetivo
  - Uma função booleana que nos diz se um dado estado é um estado objetivo.
- Uma função sucessor
  - Uma ação que nos leva de um estado para o outro
- (Opcional) um custo associado a cada ação
  - Sempre $\geq$ 0

Uma solução é um caminho do estado inicial a um estado objetivo (opcionalmente com o menor custo total).
